Instrukcja obsługi aplikacji.

Aby uruchomić aplikację należy otworzyć skrót o nazwie: Dynamiczne struktury danych,
lub wejść w folder dist\main gdzie znajduje się plik, do którego prowadzi powyższy skrót, a nazywa się: main.exe

W razie, gdyby zapora systemu Windows lub program antywirusowy oznaczył plik main.exe jako potencjalne zagrożenie,
należy zezwolić na uruchomienie tego pliku lub wyłączyć w opcjach ochronę w czasie rzeczywistym.
Problem ten został opisany w rozdziale "3.11. Problemy i ich rozwiązania" w pracy
i nie istnieje prosty sposób jego rozwiązania.

Innym możliwym sposobem otworzenia aplikacji jest wczytanie jej do środowiska PyCharm lub Virtual Studio
i uruchomienie pliku głównego programu o nazwie: main.py,
wiąże się to jednak z koniecznością posiadania środowiska oraz poprawnym skonfigurowaniu.
